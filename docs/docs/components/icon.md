# Icon 图标

## Attributes

| 参数      | 说明                           | 类型   | 可选值                | 默认值 |
| --------- | ------------------------------ | ------ | --------------------- | ------ |
| icon      | icon 的 class                  | string | ——                    | ——     |
| fontClass | FontClass 用于自定义 icon 集合 | string | ——                    | ——     |
| color     | icon 的颜色                    | string | ——                    | ——     |
| size      | icon 的字体大小                | string | ——                    | ——     |
| opacity   | icon 的透明度                  | string | 0-1（仅包含一位小数） | ——     |
